我的技术生态

大概在2019年的时候我开始使用lerna打造自己的技术生态链,因为当时我在不同的项目组之间工作,同事跟我反应项目之间的操作好像缺乏一致性,也就是我做项目有自己的一套独特架构,但是这种架构没有办法标准的复刻给另一个人,每一次启动类似项目的时候都要重新造轮子,所以我便开始打造属于自己的技术生态圈,希望后期可以建立一个自己的技术生态王国.
