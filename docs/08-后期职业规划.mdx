# 后期职业规划

作为一名nodejs开发者来讲,后期可以往两个方面进行发展  

针对中国行情来看,目前主流的技术方向都是偏向应用级开发,很少有底层开发特别厉害的企业,从业务的潜力来估算的话我偏向于区块链
所以后期我会朝区块链方向进行发展,当然区块链也有自己的底层技术,作为一门以go语言为主导的技术也方便我们这些nodejs开发者进行转型,
当然,转型的过程都是充满着挑战的,但我相信,挑战和机遇是并存的.  

然后从一名全栈工程师的角度来看,后期也可以往独立开发的方向进行发展,比如开发一些主题明确的独立内容站,在这里也要向大家推广一下
独立开发的社区([w2solo](https://w2solo.com/)),以及我在youtube上订阅的独立开发者频道[程序员的乌托邦](https://www.youtube.com/channel/UCm5Z1WsCPhxl_NznRK0tvEw)
当然独立开发者面临的挑战同样也是严峻的,因为你要自己负责SEO等繁琐的事情
